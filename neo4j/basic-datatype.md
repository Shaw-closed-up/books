# Neo4j CQL数据类型

这些数据类型类似于Java语言。它们用于定义节点或关系的属性。

Neo4j CQL支持以下数据类型-

| 序号 |    CQL数据类型     |               用法                |
| :--: | :----------------: | :-------------------------------: |
|  1   |   Boolean布尔型    | 它用于表示布尔文字：true，false。 |
|  2   |      byte字节      |        它用来表示8位整数。        |
|  3   |    short短整型     |       它用于表示16位整数。        |
|  4   |      int整型       |       它用于表示32位整数。        |
|  5   |     long长整型     |       它用于表示64位整数。        |
|  6   |    float浮点型     |      它用于表示32位浮点数。       |
|  7   | double双精度浮点型 |      它用于表示64位浮点数。       |
|  8   |      char字符      |        用于表示16位字符。         |
|  9   |    String字符串    |        它用来表示字符串。         |
