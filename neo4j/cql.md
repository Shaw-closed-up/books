# Neo4j CQL简介

CQL代表Cypher Query Language。就像Oracle数据库具有查询语言SQL一样，Neo4j也具有CQL作为查询语言。

### Neo4j CQL

- 是Neo4j图形数据库的查询语言。
- 是一种声明式模式匹配语言。
- 遵循类似SQL的语法。
- 语法非常简单，格式易于理解。

### 与SQL比较起来:

- Neo4j CQL具有执行数据库操作的命令。
- Neo4j CQL支持许多子句，例如WHERE，ORDER BY等，以轻松的方式编写非常复杂的查询。
- Neo4j CQL支持某些功能，例如字符串，聚合。除了它们，它还支持一些关系函数。
