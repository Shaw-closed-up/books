# 如何学习本课程

## 课程简介

- 课程名称：《动手学MySQL》
- 课程编号：B20
- 所需基础：[Linux](/linux),[Vim](/vim)
- 学习周期：2周（以每天投入2小时左右时间进行学习估算）
- 学习形式：在线互动[课程使用帮助](/aboutus/help.html)
- 面向群体：本教程针对的是MySQL基础方面的知识，适合各类开发中会涉及到数据库应用的同学。
- 考核标准：能理解课程内容，并独立完成课程中的示例和作业
- 学习成果：熟悉SQL语言，了解MySQL的基础知识，可以实操完成数据库常规任务。
