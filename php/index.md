# 如何学习本课程

## 课程信息

- 课程名称：《动手学PHP》
- 课程编号：B41
- 所需基础：[Linux](/linux),[Vim](/vim),[HTML](/html)
- 学习周期：3周（以每天投入2小时左右时间进行学习估算）
- 学习形式：在线互动[课程使用帮助](/aboutus/help.html)
- 面向群体：希望能使用PHP进行快速搭建网站的同学
- 学习成果：了解PHP基础知识，为使用PHP搭建CMS,企业网站打下基础
