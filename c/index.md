# 如何学习本课程
## 课程简介
- 课程名称：《动手学C语言》
- 课程编号：B10
- 所需基础：[Linux](/linux),[Vim](/vim)
- 学习周期：8周（每天投入2小时左右时间）
- 学习形式：在线互动[课程使用帮助](/aboutus/help.html)
- 面向群体：本教程针对的是想学习基础编程语言C的同学
- 考核标准：能理解课程内容，并独立完成课程中的示例和作业
- 学习成果：熟悉C语法，高级特性。可以看懂C进行独立编程。
