# 如何学习本课程

## 课程信息

- 课程名称：《动手学Redis》
- 课程编号：B21
- 所需基础：[Linux](/linux),[Vim](/vim)
- 学习周期：1周（以每天投入2小时左右时间进行学习估算）
- 学习形式：在线代码框互动[课程使用帮助](/help/)
- 面向群体：提升数据库技能，了解NoSQL的同学。
- 学习成果：熟练掌握Redis的基础应用及操作
