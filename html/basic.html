HTML 基础

不要担心本章中您还没有学过的例子，

您将在下面的章节中学到它们。

HTML 标题

HTML 标题（Heading）是通过<h1> - <h6> 标签来定义的.

<h1>这是一个标题</h1>
<h2>这是一个标题</h2>
<h3>这是一个标题</h3>

在浏览器中显示

这是一个标题

这是一个标题

这是一个标题

HTML 段落

HTML 段落是通过标签<p> 来定义的.

<p>这是一个段落。</p>
<p>这是另外一个段落。</p>

在浏览器中显示

这是一个段落。

这是另外一个段落。

HTML 链接

HTML 链接是通过标签<a>来定义的.

<a href="https://www.w3c.org">这是一个链接</a>

在浏览器中显示

这是一个链接

提示:在 href 属性中指定链接的地址。

HTML 图像

HTML 图像是通过标签 来定义的.

<img src="a.gif">

在浏览器中显示

注意： 图像的名称和尺寸是以属性的形式提供的。

HTML 如何查看源代码

你是否看过一些网页然后惊叹它是如何实现的。

如果您想找到其中的奥秘，只需要单击右键，然后选择"查看源文件"（IE）或"查看页面源代码"（Firefox），其他浏览器的做法也是类似的。这么做会打开一个包含页面 HTML 代码的窗口。
