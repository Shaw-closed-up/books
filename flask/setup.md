# Flask 环境安装及配置 			

## 本课程在线环境的安装

```bash
#验证Python及版本
python --version
```

```bash
#更新源及安装python3 pip
pip install flask -i https://pypi.tuna.tsinghua.edu.cn/simple/ 
```

```bash
#验证flask安装
python -m flask --version
```

当有版本提示时，即证明您已经在您当前的云环境中安装成功，您就可以开始进行本课程下一步的学习了。

**注：当云环境的生命周期失效后，需要重新进行安装。**
