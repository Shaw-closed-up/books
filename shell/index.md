# 如何学习本课程

## 课程简介

- 课程名称：《动手学Shell》
- 课程编号：B04
- 所需基础：[Linux](/linux),[Vim](/vim)
- 学习周期：1周（以每天投入2小时左右时间进行学习估算）
- 学习形式：在线互动[课程使用帮助](/aboutus/help.html)
- 面向群体：本课程针对的希望能使用Linux进行高效批量处理任务的同学。
- 考核标准：能理解课程内容，并独立完成课程中的示例和作业
- 学习成果：熟悉Shell相关概念，能使用Shell完成基于命令行的批量自动化任务。
