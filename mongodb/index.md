# 如何学习本课程

## 课程简介

- 课程名称：《动手学MongoDB》
- 课程编号：B22
- 所需基础：[Linux](/linux),[Vim](/vim)
- 学习周期：2周（以每天投入2小时左右时间进行学习估算）
- 学习形式：在线命令行互动[课程使用帮助](/help/)
- 面向群体：本教程针对的希望了解文档型NoSQL数据库的的同学。
- 考核标准：能理解课程内容，并独立完成课程中的示例和作业
- 学习成果：熟悉MongoDB语言，了解MongoDB的基础知识，可以实操完成觉的数据库任务。
