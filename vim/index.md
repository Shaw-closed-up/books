# 如何学习本课程

## 课程简介

- 课程名称：《动手学Vim》
- 课程编号：B03
- 所需基础：[Linux](/linux)
- 学习周期：2天（以每天投入2小时左右时间进行学习估算）
- 学习形式：在线互动[课程使用帮助](/aboutus/help.html)
- 面向群体：本课程面向刚开始使用Linux起步，希望了解并掌握终端环境下编辑器的使用的同学。
- 考核标准：能理解课程内容，并独立完成课程中的示例和作业
- 学习成果：了解Vim工作模式，熟练使用Vim完成文本编辑任务，为以后的课程和工作打下基础。
