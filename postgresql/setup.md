# PostgreSQL 环境安装配置

## 本课程在线环境的安装

### 安装postgresql环境

```bash
#更新源及安装postgresql
apt update && apt install postgresql postgresql-contrib -y

#启动postgresql服务
service postgresql start

#进入postgresql
#postgresql默认不允许使用root用户登陆，因此需要使用su命令进行切换用户
su - postgres && psql
```
